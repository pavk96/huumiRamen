## 판매 전략

1. 기존 가게는 오프라인 판매 및 배달만 하고 네이버에 단순 등록되어 있다.
   네이버 모두(https://huwoomiramen.modoo.at/)
2. 일단 웹 사이트를 먼저 준비한다.
3. 택배 배달이 가능하도록 통신판매업 신고를 위한 포장 기술 및 서류를 사장님과 같이 준비한다.
4. 준비된 웹 사이트를 광고하고 또 광고한다.
5. 맛있는 라멘을 사람들이 먹는다.
